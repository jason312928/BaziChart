# Support / 使用帮助

使用问题和可复现的 Bug 请提交 GitHub Issue。提交前请先阅读 README，并搜索是否已有相同问题。

排盘结果争议需要同时说明输入数据、预期结果和采用的计算规则。无法公开的数据请替换为不涉及真实个人的等价命例。

For usage questions and reproducible bugs, open a GitHub issue after checking the README and existing issues. Calculation reports should include the input, expected result and convention used. Replace private birth records with a non-personal equivalent test case.
