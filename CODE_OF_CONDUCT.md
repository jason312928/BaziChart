# 行为准则 / Code of Conduct

本项目欢迎不同经验、背景和观点的参与者。请保持尊重，围绕代码、文档和可验证结果讨论。

不可接受的行为包括人身攻击、骚扰、歧视、公开他人隐私信息，以及在明确要求停止后继续进行不受欢迎的沟通。

维护者可以编辑或移除不当内容，并限制持续破坏社区协作的参与者。需要私下报告行为问题时，请通过 GitHub Security Advisory 联系维护者，不要在公开 Issue 中披露当事人的个人信息。

---

This project welcomes contributors with different levels of experience, backgrounds and viewpoints. Be respectful and keep discussion focused on code, documentation and verifiable results.

Personal attacks, harassment, discrimination, publishing private information, and continuing unwanted contact after being asked to stop are not acceptable.

Maintainers may edit or remove inappropriate content and restrict participants who repeatedly disrupt collaboration. Use a GitHub Security Advisory for private conduct reports and do not disclose personal information in a public issue.
