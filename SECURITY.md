# Security Policy

## Supported Version

Security fixes are applied to the latest version on the default branch.

## Reporting a Vulnerability

Please use GitHub's **Report a vulnerability** feature to open a private Security Advisory. Do not include real birth records or other personal data in the report.

Include:

- affected version or commit;
- impact and realistic attack scenario;
- reproduction steps or a minimal proof of concept;
- any suggested mitigation.

Please allow a reasonable period for investigation before public disclosure.

## 安全问题报告

请使用 GitHub 的 **Report a vulnerability** 功能创建私密 Security Advisory，不要在公开 Issue 中披露漏洞，也不要附带真实命例或个人资料。请说明受影响版本、实际影响、复现步骤和可行的修复建议。
